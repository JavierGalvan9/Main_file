The repository contains the following files:

1) Raster_loading.py 
This script allows for loading the raster data, using TOA corrections and pansharpening the raster.

2) Preprocessing.py for preprocessing the raster data
3) Xylella_detection.py for detection of Xylella based on the raster data and the qPCR tests

Code required for:

Tree identification based on NDVI threshold.
Tree segmentation based on Watershed algorithm.
Qpcr test labelling
Machine learning algorithms applied to the identification of X. fastidiosa.
